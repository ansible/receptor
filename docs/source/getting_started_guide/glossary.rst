########
Glossary
########

The following is a list of the definitions elsewhere for receptor.

.. glossary::

    Backend peers
        A node connected to another through receptor backends

    Control node
        A node running the receptor control service

    Control service
        A service accepting commands for a receptor node

    Netceptor peers
        A receptor node directly connected to another receptor node

    Receptor backends
        A service using a network protocol to connect one receptor node to another

    Receptor node
        A node running receptor
